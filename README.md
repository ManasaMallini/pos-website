#  QuickPOS - Point of Sale System

## Project Overview

QuickPOS is a modern **Point of Sale (POS) web application** built using Flask, HTML, CSS, and JavaScript.
It helps businesses manage sales, inventory, and procurement efficiently.



##  Features

###  Sales & Billing

* Add products to cart
* Auto bill calculation (subtotal, tax, total)
* Receipt generation
* Smooth UI with animations

###  Inventory Dashboard

* Total stock tracking
* Stock value & profit display
* Interactive charts using Chart.js
* Modern dashboard UI

###  Procurement

* Product catalog
* Supplier management
* Order tracking
* Search functionality



## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **Charts:** Chart.js
* **Database:** (Optional - can integrate Supabase / MySQL)

##  How to Run the Project

1. Clone the repository


git clone https://github.com/your-username/quickpos.git


2. Install dependencies


pip install flask flask-cors


3. Run the app


python app.py


4. Open in browser


http://127.0.0.1:5000/
